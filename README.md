# Exploring and Visualizing a Simple Dataset

## 🌸 Iris Dataset Exploration and Visualization

## 🎯 Objective
Explore and visualize the Iris dataset to understand its structure, summarize key information, and identify patterns using Python.

## 🛠️ Approach
- Loaded and inspected the dataset using Pandas.
- Performed data summarization and checked for missing values.
- Created visualizations including:
  - Scatter Plots
  - Histograms
  - Box Plots
  - Correlation Heatmap
- Used Matplotlib and Seaborn for data visualization.

## 📊 Results & Insights
- The dataset contains **150 samples** and **5 columns** with no missing values.
- Petal features show a strong correlation and clearly separate the three Iris species.
- Visualizations helped understand feature distributions and detect potential outliers.

## 🧰 Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
